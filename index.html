Тег Пояснення
<html> Кореневий елемент HTML-документу.
<head> Містить метадані документа, такі як теги <meta>, <title>, та підключення стилів.
<meta> Метаінформація про сторінку, включаючи кодування, масштабування тощо.
<style> Вбудовані стилі CSS для елементів сторінки.
<body> Основний контейнер для вмісту сторінки.
<header> Контейнер для заголовку сторінки.
<h1> Головний заголовок сторінки.
<h5> Заголовок п’ятого рівня.
<p> Параграф тексту.
<blockquote> Цитата, виділена від основного тексту.
<i> Тег для курсиву.
<ul> Ненумерований список.
<li> Елемент списку.
<a> Гіперпосилання на інші сторінки або ресурси.
<img> Вставка зображення на сторінку.
<table> Контейнер для таблиці.
<thead> Група заголовків стовпців таблиці.
<tbody> Основна частина таблиці.
<tfoot> Нижня частина таблиці.
<tr> Рядок таблиці.
<th> Заголовкова комірка таблиці.
<td> Звичайна комірка таблиці.
Це реальні теги, використані на сторінці портфоліо.
